Para fazer essa mágica acontecer, usei o "trio de ferro" do Python para interfaces:

* **CustomTkinter:** O responsável por esse visual moderno (adeus, botões cinzas do Windows 95!).
* **Pillow (PIL):** A "mágica" que faz as imagens aparecerem sem perder a qualidade.
* **Python:** O cérebro por trás de tudo.
